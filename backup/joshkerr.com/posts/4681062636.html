Mars Edit for power users - (Markdown)

You are a Mars Edit user and you want to start writing your posts in Markdown but you aren’t sure where to start. You want the preview feature to show the actual theme on your blow. You need quick access to macros but aren’t sure how to set them.

If you fit into any of these categories, this post will help you. I’ve managed to configure a good workflow around MarsEdit that makes the tedium of writing blog entries rather easy. Or easier, you still have to write the content.

Using Markdown

The first thing you need to do to write in Markdown is to tell MarsEdit to default to the HTML edit view.

Just check the option for “HTML Text.” This will make MarsEdit default to the HTML mode which means your Markdown syntax doesn’t get ruined in the visual editor.

Next go into the settings for your blog and change the Default Preview Filter to Markdown.

Customize the Macros

I created custom macros to create links, references, blockquote, etc.. You can get to the customize menu from the “Format” toolbar. Just choose “edit” and create a bunch of macros. This really speeds up adding images, links, etc..

Conclusion
