Post to Tumblr from terminal

If you use Tumblr and you want to post articles from the command-line, there is a great Ruby gem that allows you to do this. Its called Tumblr-rb and you can install it by issuing:

gem install tumblr-rb

It is very easy to use and it certainly solves the problem of posting articles via a shell script.
