Jetbrains releases Objective C IDE early preview

I’m not sure what Jetbrains is thinking releasing a competitor IDE to Xcode. Sure, they do have some great features baked into their other editors and it would be nice to have some of them in XCode, but there is no chance that anything they produce will come close to the power of Xcode.

They are calling it appCode and it has the following features:

  • An IDE with really smart code editing
  • Runes on OSX Intel boxes only
  • Integrates with interface builder
  • can run iOS simulator
  • Has a debugger with evaluate expression
  • Already has ‘change signature’ and ‘extract method’
  • Supports subversion, git, perforce and cvs
  • Oh and really intelligent code completion

Most of these sound like stuff that Xcode 3 did. With Xcode 4 you can do a lot more than this. Why would I want to run Interface Builder in a separate app when I can have it combined wth Xcode 4?

I would not bet on this losing horse.
