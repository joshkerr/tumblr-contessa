Wordpress + Markdown quicktags = bliss

If you are authoring posts in Markdown and you are a Wordpress user, you NEED this plugin. It is called Markdown QuickTags and creates a Markdown specific editor for writing your Wordpress posts from within the standard editor.

The editor has a lot of shortcuts that make writing in Markdown a breeze.

  • Toolbar with quick links for creating links
  • Ordered and unordered lists
  • Blockquotes
  • Horizontal rules
  • And Wordpress more code

It also has the capability to format existing HTML code and transforming it into Markdown.

Lastly, it has a full screen editor and preview mode making it really easy to write from your web browser.

The editor is supposedly very BETA with the author only having tested on Wordpress 3.0+ and only on a mac. I’m using both, so it might break if you aren’t on that specific platform.
