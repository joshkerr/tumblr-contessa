Vico - Mac text editor

There is a new Mac text editor in the mix and its name is Vico. Vico is the result of a personal itch from the author who wanted a beautiful modern Mac editor with the power and feature set of Textmate and VI.

Vico is clearly inspired by a lot of great software and in a lot of ways captures some of those innovations and then improves upon them. For example, Vico supports Bundles from Textmate. But rather than manually downloading them, it provides a built in plugin manager. It is so much better than how Textmate does it.

Another nice feature in Vico is the built in Explorer window. You can easily activate it via a button on the toolbar on top and make it go away when you don’t need it.

Lastly, Vico has built in SFTP support. This is actually an awesome feature because it lets me work on blog files without needing to use a separate program.

Vico is in development so you’ll need to update it with a new version every week or so. (It expires after a few days.) It should be released soon as it is very far along in its development. It works great for me.
