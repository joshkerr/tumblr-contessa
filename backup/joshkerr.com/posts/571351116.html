Mars Edit 3.0

The new Mars Edit blog software for the Mac is really cool.  I’m loving the rich text editor.  This was my official blog software for the Mac, mainly because Windows Live Writer doesn’t come in a Mac version and I used it reluctantly because it was missing some features that WLW had.  Now version 3.0 of Mars Edit has those features and I’m happily using it now.  Thanks!
