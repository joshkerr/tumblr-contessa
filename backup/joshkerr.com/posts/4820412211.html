Firebug vs Chrome Web Dev

I love Google Chrome, and I love that it has web development tools built into it.  If you right click on a web page you get the option to “inspect element.”  This pulls up a great tool that will open the curtain to the structure of the currently displayed website.  Firefox also has a great tool called Firebug, but it doesn’t ship with the browser, instead you will need to download a plugin.

With the soon be released Firefox 4, I’ve been using Firebug and I think I might like it better than the Google Chrome counterpart.
